# Tiny-Lexical-Analyzer
Tiny language lexical analyzer using C#

A small project for **lexical analysis** of text using a custom programming setup.

## Description
This project reads text files and converts them into **tokens** to understand their structure.  
Great for learning and exploring the basics of **Lexical Analysis**.
